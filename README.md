# WebbWizards Todo App

![CI](https://github.com/Grahnnen/Webbwizards/actions/workflows/ci.yml/badge.svg)
![codecov](https://codecov.io/gh/Grahnnen/WebbWizards/branch/dev/graph/badge.svg)


A simple but structured **Todo application built with vanilla JavaScript**, focusing on clean architecture, state management, and testable DOM logic.

This project is part of a learning exercise in modern frontend development without frameworks.

---

## Features

- Add, remove, and view todos
- Todos persisted using `localStorage`
- Date-based filtering of todos
- Accessible modal for creating todos
- Keyboard-accessible date selector
- DOM rendering separated from business logic
- Unit-tested DOM behavior using Jest and Testing Library

---

## Tech Stack

- HTML
- CSS
- JavaScript (ES Modules)
- Jest
- @testing-library/dom

---

## Accessibility

Keyboard navigation supported for date selection

Semantic HTML elements

ARIA labels where appropriate

---

## Learning Goals

Understand DOM-driven architecture

Practice separation of concerns

Write testable JavaScript without frameworks

Gain confidence in Jest + DOM Testing Library

Work with browser APIs such as localStorage
